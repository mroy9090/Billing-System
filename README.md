# SysCraft

A high-performance, low-level system application developed in **C** programming language. This project demonstrates strong understanding of memory management, data structures, and algorithm optimization.

---

## 🚀 Features

* 🧠 Efficient use of pointers and dynamic memory
* ⚡ Fast execution and low resource consumption
* 📊 Modular code structure with reusable components
* 🧪 Unit tested core functionality
* 📁 Input/Output file handling
* 🔄 Loop and conditional logic usage
* 🧩 Custom data structures (linked list, queue, stack, etc.)

---

## 🛠️ Tech Stack

| Component        | Description             |
| ---------------- | ----------------------- |
| Language         | C                       |
| Compiler         | GCC or Clang            |
| Build Tool       | Makefile (optional)     |
| OS Compatibility | Linux / Windows / macOS |

## ⚙️ Installation & Running

### Compile Manually:

```bash
gcc -o my_program main.c utils.c -Wall
./my_program
```

### Or Use Makefile:

```bash
make
./my_program
```
